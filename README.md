<h2 id="running">Running the application 🎲</h2>
 
```bash
# Clone the repository
$ git clone https://github.com/HugoCortezL/user-auth-frontend
 
# Access the folder
$ cd person-crud-frontend
 
# Install all dependencies
$ npm install
 
# Start the service
$ npm run dev
```
